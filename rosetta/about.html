Project Rosetta

A Rainworld Linguistics Project

What is it?

Project Rosetta is a community run attept at cracking the language used in the game Rainworld. We are trying to use context clues, approximate translations and a lot of time to find out exactly what each symbol means. We try to make sure each meaning fits as well as we can before adding it here because it is absolutely imparative to not make handwavy guesses at the meanings and have to be provable beyond a reasonable doubt! Our end goal is to be able to get to a point we can comfortably explain everything in rainworlds language as well as being able to use it should we want to.

Why Rosetta?

In linguistics there is something known as the Rosetta stone it was the key to decyphering the ancient egyption scripts because it offered translations for a decree of Ptolemy V Epiphanes. Our goal is to use the approximate translations and contexts given by the developers to do the same thing here!
